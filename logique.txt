📋 GUIDE COMPLET DES COMBINAISONS TYPE + FORMULE - LOTERIE NATIONALE
🎯 TYPE : FIRST OU ONE BK
First BK + Directe
Numéros joués : 1 seul numéro
Condition de gain : Le numéro joué doit se retrouver PARMI les numéros tirés (peu importe la position)
Exemple : Je joue le 15, si le 15 sort en 1ère, 2ème, 3ème, 4ème ou 5ème position → Je gagne
First BK + Position1
Numéros joués : 1 seul numéro
Condition de gain : Le numéro joué doit être exactement le PREMIER numéro tiré
Exemple : Je joue le 15, je gagne seulement si le 15 est tiré en 1ère position
First BK + Position2
Numéros joués : 1 seul numéro
Condition de gain : Le numéro joué doit être exactement le DEUXIÈME numéro tiré
First BK + Position3
Numéros joués : 1 seul numéro
Condition de gain : Le numéro joué doit être exactement le TROISIÈME numéro tiré
First BK + Position4
Numéros joués : 1 seul numéro
Condition de gain : Le numéro joué doit être exactement le QUATRIÈME numéro tiré
First BK + Position5
Numéros joués : 1 seul numéro
Condition de gain : Le numéro joué doit être exactement le CINQUIÈME numéro tiré
🎯 TYPE : NAP
NAP + NAP3
Numéros joués : Exactement 3 numéros
Condition de gain : LES 3 numéros joués doivent TOUS se retrouver parmi les numéros tirés (ordre non important)
Exemple : Je joue [12, 35, 67], je gagne si 12 ET 35 ET 67 sortent tous (peu importe leurs positions)

📝 NOTES IMPORTANTES
Double Chance :
Win : Les 5 premiers numéros tirés (60% des gains)
Machine : Les 5 derniers numéros tirés (40% des gains)
Pour gagner en double chance, il suffit que la condition soit remplie sur Win OU sur Machine (pas les deux)
Types Automatiques :
Double Number : Le joueur ne choisit jamais ses numéros, ce sont toujours les 8 doubles
Anagramme Simple : Le joueur ne choisit jamais ses numéros, ce sont toujours les 37 binômes prédéfinis
